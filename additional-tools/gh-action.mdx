## Overview
Our GitHub Action utilizes the Ditto CLI to automatically create pull requests on GitHub with the latest text updates in any repository where the action is triggered.
Check it out [here](https://github.com/marketplace/actions/ditto-github-action).

## Additional Information
For setup instructions and example workflows, see the instructions listed [on the Github Action](https://github.com/marketplace/actions/ditto-github-action).
